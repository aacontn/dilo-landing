## Lo que preguntarías antes de instalar

¿Por qué es gratis? ¿Cuál es el truco?

No hay truco: es open source (MIT) y el modelo corre en tu compu, así que nadie paga servidores por ti. Dilo es un fork con dirección propia de Handy, de CJ Pais. Si te sirve, compártelo o contribuye código — esa es toda la "monetización".

¿A dónde va mi voz?

A ningún lado. El audio se procesa localmente y se descarta. Sin internet también funciona — ponle modo avión y dicta igual.

¿Qué tan bien entiende el español?

Los modelos recomendados están entrenados con español real y aguantan spanglish técnico ("el deploy", "los hooks"). Además puedes agregar tus palabras propias — nombres, siglas, jerga de tu equipo — para que no se las invente.

¿Por qué mi sistema me advierte al instalarlo?

Los binarios van sin firma de código todavía (el certificado de Apple cuesta US$99/año; está en el roadmap). En macOS: clic derecho → Abrir. En Windows: Más información → Ejecutar de todas formas. El código es público — compílalo tú mismo si prefieres.

¿Qué necesita mi compu?

Con 8 GB de RAM y el modelo liviano (218 MB) andas perfecto. Con 16 GB, el modelo streaming te muestra el texto en vivo mientras hablas. macOS (Apple Silicon o Intel), Windows 10/11 y Linux. No necesitas GPU.

¿Sirve para dictar código?

Para prompts, commits, issues, comentarios y docs es oro. Para código carácter-por-carácter el teclado sigue ganando — por eso Dilo convive con él en vez de reemplazarlo.
